GIỚI THIỆU CHƯƠNG

5. CSS Nâng cao

Chương này sẽ giới thiệu các kỹ thuật CSS nâng cao giúp tạo ra các layout phức tạp, hiệu ứng đẹp mắt và tương tác linh hoạt với người dùng.

Mục tiêu học tập

  • Hiểu và áp dụng được các thuộc tính hiển thị và định vị nâng cao
  • Làm chủ Flexbox và Grid System để tạo layout hiện đại
  • Tạo được hiệu ứng chuyển động và animation
  • Sử dụng thành thạo các lớp giả và phần tử giả
  • Áp dụng các thuộc tính CSS nâng cao khác

Tầm quan trọng

CSS nâng cao là chìa khóa để tạo ra các website hiện đại, responsive và có trải nghiệm người dùng tốt. Nắm vững các kỹ thuật này giúp bạn giải quyết các bài toán layout phức tạp một cách hiệu quả.

HIỂN THỊ & VỊ TRÍ

5.1. Các thuộc tính hiển thị và định vị

Thuộc tính Mô tả Giá trị phổ biến
visibility Ẩn nhưng vẫn chiếm chỗ visible, hidden
position Vị trí phần tử static, relative, absolute, fixed, sticky
z-index Độ sâu lớp 1, 999
top, left, right, bottom Khoảng cách định vị 0, 10px
float Đẩy trái/phải left, right, none

5.1.1. visibility

Mục đích: Ẩn hoặc hiển thị phần tử nhưng vẫn giữ chỗ (khác với display: none)

Ví dụ visibility
<p style="visibility: hidden;">Ẩn nhưng vẫn chiếm chỗ</p>
<p>Đoạn văn bình thường</p>

5.1.2. position

Mục đích: Thiết lập cách phần tử được định vị trên trang

Ví dụ position
<div class="box static">Static</div>
<div class="box relative">Relative</div>
<div class="container">
  <div class="box absolute">Absolute</div>
</div>
<div class="box fixed">Fixed</div>

/* CSS */
.box {
  padding: 10px;
  margin: 10px;
  color: white;
}
.static {
  position: static;
  background: gray;
}
.relative {
  position: relative;
  top: 10px;
  left: 20px;
  background: blue;
}
.container {
  position: relative;
  height: 150px;
  border: 1px solid black;
}
.absolute {
  position: absolute;
  top: 20px;
  left: 30px;
  background: green;
}
.fixed {
  position: fixed;
  bottom: 10px;
  right: 10px;
  background: red;
}

Lưu ý quan trọng

Thuộc tính absolute cho phép căn chỉnh một phần tử dựa vào thẻ bao ngoài có giá trị của position là relative. Nếu không có phần tử cha nào có position: relative, nó sẽ định vị theo thẻ <body>.

5.1.3. z-index

Mục đích: Xác định lớp xếp chồng (phần tử nào nằm trên phần tử nào)

Ví dụ z-index
.box1 {
  position: absolute;
  z-index: 2;
  background: red;
}
.box2 {
  position: absolute;
  z-index: 1;
  background: blue;
}

FLEXBOX

5.2. Flexbox - Bố cục linh hoạt

5.2.1. Flex Container - Thuộc tính áp dụng cho phần tử cha

Thuộc tính Mô tả Giá trị phổ biến
display Kích hoạt chế độ Flexbox flex, inline-flex
flex-direction Xác định hướng sắp xếp row, row-reverse, column, column-reverse
justify-content Căn chỉnh theo trục chính flex-start, center, flex-end, space-between, space-around
align-items Căn chỉnh theo trục phụ stretch, flex-start, center, flex-end
flex-wrap Xuống dòng khi tràn nowrap, wrap, wrap-reverse
gap Khoảng cách giữa các item 10px, 1rem, 20px

5.2.2. Flex Item - Thuộc tính áp dụng cho phần tử con

Thuộc tính Mô tả Giá trị phổ biến
order Sắp xếp thứ tự xuất hiện 0, 1, 2, -1
flex-grow Tỉ lệ giãn ra khi còn chỗ trống 0, 1, 2
flex-shrink Khả năng thu nhỏ khi thiếu chỗ 0, 1
flex-basis Kích thước ban đầu auto, 100px, 20%
flex Viết tắt (grow, shrink, basis) 1 1 auto, 0 0 200px
Ví dụ Flexbox hoàn chỉnh
<!-- HTML -->
<div class="container">
  <div class="item item1">1</div>
  <div class="item item2">2</div>
  <div class="item item3">3</div>
</div>

/* CSS */
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 10px;
  height: 200px;
  background: #f8f9fa;
  padding: 20px;
}
.item {
  background: #3498db;
  color: white;
  padding: 20px;
  border-radius: 5px;
}
.item1 {
  flex-grow: 1;
  order: 2;
}
.item2 {
  flex-grow: 2;
  order: 1;
}
.item3 {
  flex-grow: 1;
  order: 3;
}

GRID SYSTEM

5.3. Grid System - Hệ thống lưới

5.3.1. Thuộc tính áp dụng cho Grid Container

Thuộc tính Mô tả Ví dụ
display Kích hoạt Grid grid, inline-grid
grid-template-columns Xác định số và kích thước cột 1fr 2fr 1fr, repeat(3, 1fr)
grid-template-rows Xác định số và kích thước hàng 100px 200px, auto
gap Khoảng cách giữa các ô 10px, 1rem
grid-template-areas Đặt tên cho các vùng "header header" "sidebar content"

5.3.2. Thuộc tính áp dụng cho Grid Item

Thuộc tính Mô tả Ví dụ
grid-column Xác định vị trí theo cột 1 / 3, span 2
grid-row Xác định vị trí theo hàng 1 / 4, span 3
grid-area Gán cho vùng đã đặt tên header, sidebar
Ví dụ Grid System hoàn chỉnh
<!-- HTML -->
<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="content">Main Content</div>
  <div class="ads">Ads</div>
  <div class="footer">Footer</div>
</div>

/* CSS */
.container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: 80px auto 60px;
  grid-template-areas:
    "header header header"
    "sidebar content ads"
    "footer footer footer";
  gap: 10px;
  height: 400px;
}
.header {
  grid-area: header;
  background: lightblue;
}
.sidebar {
  grid-area: sidebar;
  background: lightgray;
}
.content {
  grid-area: content;
  background: white;
}
.ads {
  grid-area: ads;
  background: lightyellow;
}
.footer {
  grid-area: footer;
  background: lightgreen;
}

HIỆU ỨNG & HOẠT HÌNH

5.4. Tạo hiệu ứng và animation

5.4.1. transition - Hiệu ứng chuyển tiếp

Định nghĩa: transition giúp tạo hiệu ứng mượt mà khi một thuộc tính CSS thay đổi

Ví dụ transition
.box {
  width: 100px;
  height: 100px;
  background: blue;
  transition: all 0.5s ease-in-out;
}
.box:hover {
  background: red;
  width: 150px;
}

Demo transition:

Hover vào tôi!

5.4.2. transform - Biến đổi đối tượng

Định nghĩa: Cho phép dịch chuyển, xoay, phóng to/thu nhỏ phần tử

Ví dụ transform
.box {
  width: 100px;
  height: 100px;
  background: green;
  transition: all 0.3s ease;
}
.box:hover {
  transform: scale(1.2) rotate(10deg);
}

Hover vào tôi!

5.4.3. animation & @keyframes - Hoạt hình

Định nghĩa: Tạo hoạt ảnh nhiều bước thông qua @keyframes

Ví dụ animation
.box {
  width: 100px;
  height: 100px;
  background: orange;
  animation: slideIn 1s ease forwards;
}
@keyframes slideIn {
  from {
    transform: translateX(-200px);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

THUỘC TÍNH KHÁC

5.5. Các thuộc tính CSS nâng cao khác

Thuộc tính Mô tả Giá trị phổ biến
cursor Kiểu con trỏ pointer, default, text, move
overflow Xử lý nội dung tràn hidden, scroll, auto, visible
visibility Ẩn/hiện (giữ vị trí) visible, hidden, collapse
box-shadow Đổ bóng hộp 0 4px 8px rgba(0,0,0,0.2)
filter Hiệu ứng hình ảnh blur(5px), grayscale(100%)

5.5.1. cursor

Định nghĩa: Thay đổi kiểu con trỏ chuột khi di chuyển vào phần tử

Ví dụ cursor
button {
  cursor: pointer;
}
input {
  cursor: text;
}
.disabled {
  cursor: not-allowed;
}

5.5.2. overflow

Định nghĩa: Quyết định xử lý nội dung bị tràn ra ngoài phần tử chứa

Ví dụ overflow
.scroll-box {
  width: 200px;
  height: 100px;
  overflow: scroll;
}
.hidden-box {
  overflow: hidden;
}

5.5.4. box-shadow

Định nghĩa: Tạo bóng đổ bên ngoài phần tử

Ví dụ box-shadow
.card {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
  /* offsetX offsetY blurRadius color */
}

5.5.5. filter

Định nghĩa: Áp dụng hiệu ứng hình ảnh (làm mờ, đổi màu, v.v.)

Ví dụ filter
img {
  filter: grayscale(100%);
  /* Chuyển ảnh sang đen trắng */
}

.blur-img {
  filter: blur(5px);
  /* Làm mờ ảnh */
}

LỚP GIẢ & PHẦN TỬ GIẢ

5.6. Pseudo-class và Pseudo-element

5.6.1. Pseudo-class (lớp giả)

Định nghĩa: Là một từ khóa được thêm vào selector để xác định trạng thái đặc biệt của phần tử HTML

Ví dụ pseudo-class
/* Thay đổi màu khi hover */
a:hover {
  color: red;
}

/* Focus vào input */
input:focus {
  border: 2px solid blue;
}

/* Phần tử con đầu tiên */
li:first-child {
  font-weight: bold;
}

/* Phần tử con thứ n */
li:nth-child(2n) {
  background: #f0f0f0;
}

5.6.2. Pseudo-element (phần tử giả)

Định nghĩa: Cho phép tạo và định dạng một phần cụ thể của phần tử mà không cần thêm thẻ HTML mới

Ví dụ pseudo-element
/* Thêm nội dung trước tiêu đề */
h1::before {
  content: "🔥 ";
}

/* Thêm nội dung sau tiêu đề */
h1::after {
  content: " ●";
}

/* Định dạng ký tự đầu tiên */
p::first-letter {
  font-size: 2em;
  color: blue;
}

/* Định dạng dòng đầu tiên */
p::first-line {
  color: green;
}

/* Định dạng phần được chọn */
::selection {
  background: yellow;
  color: black;
}
