GIỚI THIỆU GRID SYSTEM

CSS Grid Layout là một hệ thống layout 2 chiều (hàng và cột) mạnh mẽ trong CSS3. Khác với Flexbox (1 chiều), Grid cho phép bạn tạo các layout phức tạp với sự kiểm soát chính xác cả hai chiều.

Tại sao nên dùng Grid?

Layout 2 chiều: Kiểm soát cả hàng và cột đồng thời

Responsive mạnh mẽ: Dễ dàng tạo layout thích ứng với fr unit và auto-fit

Căn chỉnh chính xác: Đặt phần tử vào bất kỳ vị trí nào trong lưới

Template areas: Định nghĩa layout bằng tên vùng trực quan

Khái niệm cơ bản

Grid vs Flexbox

Flexbox: 1 chiều, tốt cho việc sắp xếp các item trong một hàng hoặc một cột.

Grid: 2 chiều, hoàn hảo cho layout phức tạp cần kiểm soát cả hàng và cột.

Bạn có thể kết hợp cả hai: dùng Grid cho layout tổng thể và Flexbox cho các thành phần bên trong.

5.3.1.1. display: grid

Định nghĩa: Biến phần tử trở thành một grid container, cho phép sử dụng tất cả các thuộc tính của Grid Layout.

Cú pháp
.container {
  display: grid;
}

grid: Phần tử là một block-level grid container.

inline-grid: Phần tử là một inline-level grid container.

subgrid: Kế thừa grid từ container cha (chưa được hỗ trợ đầy đủ).

5.3.1.2. grid-template-columns & grid-template-rows

Định nghĩa: Xác định số lượng và kích thước các cột và hàng trong grid.

Cú pháp
.container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr; /* 3 cột: tỉ lệ 1-2-1 */
  grid-template-rows: 100px 200px; /* 2 hàng: cao 100px và 200px */
}

Đơn vị fr và hàm repeat()

fr (fraction): Đơn vị phân số, chia không gian còn lại theo tỉ lệ.

repeat(): Hàm lặp lại giá trị, giúp viết code ngắn gọn hơn.

/* Cách viết thông thường */
grid-template-columns: 1fr 1fr 1fr 1fr;

/* Dùng repeat() */
grid-template-columns: repeat(4, 1fr);

/* Kết hợp nhiều giá trị */
grid-template-columns: 200px repeat(3, 1fr) 100px;

5.3.1.4. grid-template-areas

Định nghĩa: Đặt tên cho các vùng trong grid để dễ sắp xếp và tạo layout phức tạp.

Ví dụ cú pháp
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
  grid-template-columns: 1fr 3fr;
  grid-template-rows: auto auto auto;
}

/* Gán vùng cho từng phần tử */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

  • Mỗi hàng được đặt trong dấu ngoặc kép
  • Mỗi vùng được đặt tên (không có khoảng trắng)
  • Dấu chấm (.) để chỉ ô trống
  • Các vùng có thể mở rộng nhiều ô bằng cách lặp lại tên

5.3.1.5. justify-items & align-items

Định nghĩa: Căn chỉnh nội dung bên trong các grid items theo chiều ngang (justify-items) và chiều dọc (align-items).

Cú pháp
.container {
  justify-items: center; /* căn ngang */
  align-items: stretch; /* căn dọc */
}

start: Căn về đầu trục

end: Căn về cuối trục

center: Căn giữa

stretch (mặc định): Kéo dài để lấp đầy ô

5.3.1.6. justify-content & align-content

Định nghĩa: Căn chỉnh toàn bộ grid trong container khi grid nhỏ hơn container.

Cú pháp
.container {
  justify-content: space-between; /* căn ngang */
  align-content: center; /* căn dọc */
}

Khi nào dùng?

Chỉ có tác dụng khi tổng kích thước các track (cột/hàng) nhỏ hơn kích thước container.

5.3.2.1. grid-column & grid-row

Định nghĩa: Xác định vị trí phần tử trên lưới theo hàng/cột.

Cú pháp
.item {
  /* Bắt đầu từ cột 1 đến trước cột 3 */
  grid-column: 1 / 3;
  /* Bắt đầu từ hàng 2 đến trước hàng 4 */
  grid-row: 2 / 4;
}

/* Hoặc dùng span */
.wide-item {
  grid-column: span 2; /* chiếm 2 cột */
}

5.3.2.3. Ví dụ thực tế - Layout website

Mục tiêu: Tạo layout website hoàn chỉnh với Grid System, bao gồm header, sidebar, content, ads và footer.

HTML
<div class="website-layout">
  <header class="header">Header</header>
  <nav class="sidebar">Sidebar</nav>
  <main class="content">Main Content</main>
  <aside class="ads">Advertisement</aside>
  <footer class="footer">Footer</footer>
</div>
CSS Grid Layout
.website-layout {
  display: grid;
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: 80px auto 60px;
  grid-template-areas:
    "header header header"
    "sidebar content ads"
    "footer footer footer";
  gap: 10px;
  min-height: 100vh;
}

.header { grid-area: header; background: lightblue; }
.sidebar { grid-area: sidebar; background: lightgray; }
.content { grid-area: content; background: white; }
.ads { grid-area: ads; background: lightyellow; }
.footer { grid-area: footer; background: lightgreen; }

Ưu điểm của Grid cho layout website:

  • Code ngắn gọn: Chỉ cần vài dòng CSS để tạo layout phức tạp
  • Dễ bảo trì: Thay đổi layout bằng cách sửa grid-template-areas
  • Responsive dễ dàng: Dùng media queries để thay đổi grid layout
  • Căn chỉnh chính xác: Kiểm soát hoàn toàn vị trí các phần tử
