GIỚI THIỆU FLEXBOX

Flexbox (Flexible Box Layout) là một mô hình layout trong CSS3 giúp sắp xếp, căn chỉnh và phân phối không gian giữa các phần tử trong một container, ngay cả khi kích thước của chúng không xác định hoặc thay đổi.

Tại sao nên dùng Flexbox?

Dễ dàng căn chỉnh các phần tử theo chiều ngang và dọc

Tự động phân phối không gian giữa các phần tử

Responsive tự nhiên mà không cần media queries phức tạp

Kiểm soát thứ tự hiển thị của các phần tử

Khái niệm cơ bản

5.2.1.1. display

Định nghĩa: Kích hoạt chế độ Flexbox trên phần tử cha, giúp các phần tử con tự động sắp xếp theo quy tắc linh hoạt.

Cú pháp
.container {
  display: flex;
}

flex: Phần tử là một block container có Flexbox.

inline-flex: Giống flex nhưng hành vi như inline.

Ví dụ: display

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;
}
Thử nghiệm:
1
2
3
.container {
  display: flex;
}

Kết quả:

3 div con nằm ngang cùng 1 hàng (thay vì xếp dọc như block thông thường).

5.2.1.2. flex-direction

Định nghĩa: Xác định hướng sắp xếp các item con trong container.

Cú pháp
.container {
  flex-direction: row; /* mặc định */
}

row: Theo chiều ngang trái → phải (mặc định).

row-reverse: Theo chiều ngang phải → trái.

column: Theo chiều dọc trên → dưới.

column-reverse: Theo chiều dọc dưới → trên.

Ví dụ: flex-direction: column

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;
  flex-direction: column;
}

Kết quả:

Các phần tử con xếp theo chiều dọc từ trên xuống.

5.2.1.3. justify-content

Định nghĩa: Căn chỉnh các item theo trục chính (main axis).

Cú pháp
.container {
  justify-content: center;
}

flex-start: Dồn về đầu trục.

flex-end: Dồn về cuối trục.

center: Căn giữa.

space-between: Các item giãn đều, không khoảng ở 2 bên.

space-around: Các item giãn đều, có khoảng ở 2 bên.

space-evenly: Khoảng cách đều nhau mọi phía.

Ví dụ: justify-content

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;
  justify-content: center;
}

5.2.1.4. align-items

Định nghĩa: Căn chỉnh các item theo trục phụ (cross axis) – thường là chiều dọc.

Chỉ có tác dụng khi flex-wrap: wrap được dùng!

Cú pháp
.container {
  align-items: center;
}

stretch (mặc định): Kéo dài các item theo chiều cross.

flex-start: Dồn lên đầu cross.

flex-end: Dồn về cuối cross.

center: Căn giữa theo chiều dọc.

Ví dụ: align-items

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;
  align-content: center;
}

Demo tương tác: align-items

2
3

  display: flex;
  align-items: center;
  height: 200px;
}

5.2.1.5. align-content

Định nghĩa: Căn chỉnh nhiều dòng item theo trục phụ khi chúng xuống dòng (tức là khi có nhiều dòng).

Cú pháp
.container {
  align-content: space-between;
}

Lưu ý quan trọng:

Chỉ có tác dụng khi flex-wrap: wrap được dùng và có nhiều dòng item!

5.2.2.1. order

Định nghĩa: Sắp xếp thứ tự xuất hiện của item.

Cú pháp
.item {
  order: 2;
}

Lưu ý:

Mặc định: order: 0; — số nhỏ hơn sẽ được hiển thị trước.

Ví dụ: order

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;

}
.item1 {order: 2;}
.item2 {order: 1;}
.item3 {order: 1;}

5.2.2.2. flex-grow

Định nghĩa: Xác định tỉ lệ phần tử giãn ra khi còn chỗ trống.

Cú pháp
.item {
  flex-grow: 1;
}

Lưu ý:

Mặc định: 0 – không giãn.

Ví dụ: flex-grow

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;

}
.item1 {flex-grow: 1;}
.item2 {flex-grow: 2;}
.item1 {flex-grow: 1;}

Demo tương tác: flex-grow

Item 1 (grow: 1)
Item 2 (grow: 2)
Item 3 (grow: 1)

Kết quả:

Item 2 rộng gấp đôi Item 1 nếu có chỗ trống (vì flex-grow: 2 so với 1).

5.2.2.3. flex-shrink

Định nghĩa: Xác định khả năng thu nhỏ của item khi không đủ không gian.

Cú pháp
.item {
  flex-shrink: 1;
}

Lưu ý:

Mặc định: 1 – có thể co lại.

Ví dụ: flex-shrink

HTML
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
</div>
CSS
.container {
  display: flex;

}
.item1 {flex-shrink: 1;}
.item2 {flex-shrink: 0;}
.item1 {flex-shrink: 1;}

Demo tương tác: flex-shrink

Item 1 (shrink: 1)
Item 2 (shrink: 0)
Item 3 (shrink: 1)

Giải thích:

Item 2 có flex-shrink: 0 nên sẽ không co lại khi container bị thu nhỏ, trong khi Item 1 và 3 có thể co lại.

5.2.2.3. flex-basis

Định nghĩa: Chiều rộng (hoặc cao nếu là column) ban đầu của item trước khi không gian được phân phối.

Cú pháp
.item {
  flex-basis: 100px;
}

Lưu ý:

Mặc định: auto - lấy kích thước từ width/height.

Demo tương tác: flex-basis

5.2.2.4. flex (viết tắt)

Định nghĩa: Gộp 3 thuộc tính flex-grow, flex-shrink, flex-basis thành một thuộc tính duy nhất.

Cú pháp
.item {
  flex: 1 0 150px;
}

Ý nghĩa:

  • 1: cho phép giãn (flex-grow)
  • 0: không co (flex-shrink)
  • 150px: chiều dài cơ bản (flex-basis)

Các giá trị đặc biệt:

  • flex: initial = flex: 0 1 auto
  • flex: auto = flex: 1 1 auto
  • flex: none = flex: 0 0 auto
  • flex: 1 = flex: 1 1 0%
